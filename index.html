الفصل الأول — HTML5

ما هي HTML؟

لغة HTML هي اللغة الأساسية المستخدمة في بناء صفحات الويب، وتستخدم لإنشاء العناوين والفقرات والصور والروابط والجداول.

وسوم HTML

الوسوم هي أوامر يتم كتابتها داخل أقواس لتحديد نوع المحتوى داخل الصفحة.

الصور والروابط

يتم استخدام img لإضافة الصور و a لإنشاء الروابط داخل المواقع.

الفصل الثاني — CSS3

ما هي CSS؟

لغة تستخدم لتنسيق صفحات الويب والتحكم الكامل في الألوان والخلفيات والخطوط.

الخلفيات والتأثيرات

إضافة الخلفيات والصور المتدرجة والظلال والزخارف الحديثة للموقع.

Animations

إضافة الحركات والانتقالات والتأثيرات البصرية الاحترافية.

الفصل الثالث — Responsive Design

التصميم المتجاوب

يجعل الموقع مناسبًا لجميع الأجهزة مثل الهواتف والتابلت والكمبيوتر.

Flexbox

تقنية حديثة لترتيب العناصر داخل الصفحات بسهولة ومرونة.

CSS Grid

تقنية احترافية لإنشاء تخطيطات متقدمة للمواقع الإلكترونية.

الفصل الرابع — JavaScript

ما هي JavaScript؟

لغة برمجية تستخدم لجعل الموقع تفاعليًا وديناميكيًا.

التفاعل مع المستخدم

إضافة أزرار وتأثيرات واستجابة لحركة المستخدم.

DOM

التحكم الكامل بعناصر الصفحة باستخدام JavaScript.
